# Project Overview

## Project Title

Indian Food Truck Management System

## Problem Statement

Food trucks often rely on manual methods for managing menu updates, catering requests, and truck location announcements. This creates inefficiencies and poor customer communication.

This system provides a centralized digital platform to manage these operations efficiently.

---

## Objectives

The main objectives of this system are:

* Provide a digital menu for customers
* Enable customers to request catering services
* Allow the owner to manage food truck operations
* Provide real-time truck location updates
* Offer an easy-to-use administrative interface

---

## Target Users

### Customers

People looking to:

* view menu
* locate truck
* request catering

### Owner/Admin

Responsible for:

* managing menu items
* responding to catering requests
* updating truck schedule
* configuring site settings

---

## System Scope

The system includes:

Customer interface

* Homepage
* Menu page
* Catering request page
* Truck location page

Admin interface

* Dashboard
* Menu management
* Catering inbox
* Schedule manager
* Site settings
