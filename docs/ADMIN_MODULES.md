# Admin Modules

## Admin Dashboard

The dashboard provides a centralized interface for managing all system modules.

Modules:

* **Menu Management**: CRUD operations for the daily food truck menu.
* **Catering Requests**: Live inbox for managing quotes and customer communication.
* **Truck Schedule**: Real-time updates for "Today's Stop" and "Next Stop".
* **Site Settings**: Global branding, contact information, and feature toggles.
* **Catering Menu**: Management of bulk pricing, industrial packages, and tray options.

---

## Menu Management

Functions:
* Add new menu item with tags (Veg, Spicy, Popular).
* Edit existing names, descriptions, and prices.
* Temporary availability toggle (In Stock / Out of Stock).
* Drag-and-drop reordering (if implemented).

---

## Catering Requests

Functions:
* Real-time notification of new requests.
* Status tracking: NEW -> CONTACTED -> DONE.
* Secure chat-link generation for direct customer communication.

---

## Truck Schedule Manager

Functions:
* Quick-update today's location and hours.
* Schedule upcoming stops to keep customers informed.
* Toggle "Closed" status for holidays or maintenance.
