# User Manual

This manual provides instructions for both customers using the website and administrators managing the food truck business.

---

## 👤 Customer Guide

### Exploring the Menu
1. Click **"Menu"** in the navigation bar.
2. Filter items by category (Mains, Starters, Drinks) or use the search bar.
3. Look for the ✨ **Popular** badge for house favorites.

### Ordering Catering (Professional Flow)
1. Navigate to the **"Catering"** page.
2. Select your event size (Packages for 20+ people or Trays for bulk ordering).
3. Click an item to open the **Customization Drawer**.
    - Choose "Half Tray" (feeds 10-15) or "Full Tray" (feeds 25-30).
    - Select spice levels or specific preferences.
4. Review your **Selection Summary** at the bottom of the screen.
5. Fill out the event form and submit.
6. **Important**: You will receive an email with a unique chat link. Click this link to discuss pricing and logistics directly with the owner.

---

## 🔐 Administrative Guide

### Accessing the Dashboard
1. Go to `/admin/login`.
2. Enter the secure password. 

### Module 1: Menu Management
- **Add Item**: Use the "plus" button. Ensure you set a correct `sortOrder` to control its position on the public menu.
- **Availability**: Use the "In Stock" toggle. Items marked "Out of Stock" will automatically show as unavailable on the public menu.
- **Popularity**: Mark items as "Popular" to showcase them in the "Signature Dishes" section.

### Module 2: Schedule Manager
- **Update Today**: Change the location and live hours. This reflects immediately in the homepage's "Today" section.
- **Future Stops**: Plan ahead by setting "Next Stop" location and notes.
- **Saved Locations**: Pick from previously used addresses to save time.

### Module 3: Catering Inbox
- **Inquiries**: New submissions appear at the top.
- **Messaging**: Click a request to open the chat thread. Type a response to send a message back to the customer.
- **Status Tracking**: Move requests to "CONTACTED" once you've started the dialogue, and "DONE" after the event is booked.

### Module 4: Site Settings
- **Catering Toggle**: If you are booked out, turn off "Catering Enabled". This will stop new submissions and notify customers.
- **Announcement Banner**: Enable a site-wide banner for holidays, price changes, or special events.
