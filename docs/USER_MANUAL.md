# User Manual

## Customer Guide

### Browsing the Menu
1. Visit the homepage to see "Signature Dishes".
2. Click "Menu" in the navigation to see the full daily offerings.

### Finding the Truck
1. The homepage displays "Today's Stop" with live hours.
2. Click the location text to open directions in Google Maps.

### Requesting Catering
1. Navigate to the "Catering" page.
2. Browse the catering-specific menu (Packages, Trays, Appetizers).
3. Select items to add them to your request.
4. Fill out the event details form and submit.
5. Use the provided chat link to discuss the quote with the owner.

---

## Admin Guide

### Login
1. Navigate to `/admin/login`.
2. Enter the secure admin password.

### Managing Content
1. Use the **Admin Dashboard** to pick a module.
2. To update the menu: Go to **Menu Items**, edit prices or details, and save.
3. To update location: Go to **Schedule Manager**, update the stop name, and publish.
4. To handle catering: Check the **Catering Inbox**, click a request to see details, and update status as you reach out.
